# Pivora Landing Page

Static HTML landing page ready to deploy on Vercel.

## Deploy on Vercel

1. Upload this folder or ZIP to a GitHub repository.
2. Import the repository in Vercel.
3. Keep Framework Preset as **Other**.
4. Leave Build Command empty.
5. Set Output Directory to `.` if Vercel asks.
6. Deploy.

The main page is `index.html`.
